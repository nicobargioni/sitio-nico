Parte 1 — Distribuciones de probabilidad

a) Cerrajero — reclamos de clientes

> Un cerrajero observa que 1 de cada 8 clientes realiza algún reclamo. Si en un día recibe 7 clientes, ¿cuál es la probabilidad de que exactamente 3 realicen un reclamo?

Distribución: Binomial. Hay un número fijo de ensayos ($n=7$), cada cliente reclama o no (dos resultados), los clientes son independientes y la probabilidad de reclamo es constante $p = 1/8 = 0{,}125$.

$$P(X=k)=\binom{n}{k}\,p^{k}(1-p)^{\,n-k},\qquad n=7,\;p=\tfrac18,\;k=3$$

n, p, k = 7, 1/8, 3
pa = stats.binom.pmf(k, n, p)
print(f"p = 1/8 = {p:.4f}")
print(f"P(X = 3) = C(7,3) · p^3 · (1-p)^4 = {pa:.6f}  ≈  {pa*100:.2f}%")
p = 1/8 = 0.1250
P(X = 3) = C(7,3) · p^3 · (1-p)^4 = 0.040071  ≈  4.01%

Respuesta a): la probabilidad de que exactamente 3 de los 7 clientes reclamen es ≈ 0,0401 (4,01 %).

b) Aerolínea — overbooking

> El 1 % de las personas que reservan no se presenta. La aerolínea vende 300 pasajes para un avión de 299 asientos. Suponiendo que las no presentaciones siguen una distribución de Poisson, ¿cuál es la probabilidad de que todas las personas que se presenten tengan asiento?

Distribución: Poisson. Modela la cantidad de eventos raros (*no-shows*) en 300 reservas, con

$$\lambda = n\cdot p = 300 \times 0{,}01 = 3.$$

Con 300 pasajes y 299 asientos, todos los presentes tienen asiento sólo si se presentan a lo sumo 299, es decir si hay al menos 1 no-show:

$$P(\text{todos sentados}) = P(X \ge 1) = 1 - P(X=0) = 1 - e^{-\lambda}.$$

lam = 300 * 0.01          # λ = 3 no-shows esperados
p0  = stats.poisson.pmf(0, lam)   # ningún no-show -> se presentan los 300 -> 1 sin asiento
pb  = 1 - p0                      # al menos 1 no-show -> ≤ 299 presentes -> todos con asiento
print(f"λ = n·p = 300 · 0.01 = {lam}")
print(f"P(X = 0) = e^(-λ) = {p0:.6f}")
print(f"P(todos con asiento) = 1 - e^(-λ) = {pb:.6f}  ≈  {pb*100:.2f}%")
λ = n·p = 300 · 0.01 = 3.0
P(X = 0) = e^(-λ) = 0.049787
P(todos con asiento) = 1 - e^(-λ) = 0.950213  ≈  95.02%

Respuesta b): la probabilidad de que todos los presentes tengan asiento es ≈ 0,9502 (95,02 %).

c) Cereales — peso de las cajas

> El llenado sigue una distribución Normal con media $\mu = 85$ g y desvío $\sigma = 2{,}5$ g.

> 1. $P(X > 88{,}4)$

> 2. $P(86{,}9 < X < 87{,}5)$

Distribución: Normal. Se tipifica con $z = \dfrac{x-\mu}{\sigma}$.

mu, sigma = 85, 2.5

# 1) P(X > 88.4)
x1 = 88.4
z1 = (x1 - mu) / sigma
p_mayor = 1 - stats.norm.cdf(x1, mu, sigma)
print(f"P(X > 88.4):  z = (88.4-85)/2.5 = {z1:.2f}")
print(f"  P(X > 88.4) = 1 - Φ(z) = {p_mayor:.6f}  ≈  {p_mayor*100:.2f}%\n")

# 2) P(86.9 < X < 87.5)
xa, xb = 86.9, 87.5
za, zb = (xa-mu)/sigma, (xb-mu)/sigma
p_entre = stats.norm.cdf(xb, mu, sigma) - stats.norm.cdf(xa, mu, sigma)
print(f"P(86.9 < X < 87.5):  z1 = {za:.2f}, z2 = {zb:.2f}")
print(f"  P(86.9 < X < 87.5) = Φ(z2) - Φ(z1) = {p_entre:.6f}  ≈  {p_entre*100:.2f}%")
P(X > 88.4):  z = (88.4-85)/2.5 = 1.36
  P(X > 88.4) = 1 - Φ(z) = 0.086915  ≈  8.69%

P(86.9 < X < 87.5):  z1 = 0.76, z2 = 1.00
  P(86.9 < X < 87.5) = Φ(z2) - Φ(z1) = 0.064972  ≈  6.50%

Respuesta c):

  • $P(X > 88{,}4) \approx 0{,}0869$ (8,69 %).
  • $P(86{,}9 < X < 87{,}5) \approx 0{,}0650$ (6,50 %).

2.5 Reducción de dimensionalidad (PCA) para visualizar

Con 6 variables no se pueden graficar los clusters directamente. Se aplica PCA para proyectar a 2 componentes y ver si los grupos quedan separados.

pca = PCA(n_components=2)
comp = pca.fit_transform(X_std)
df["PC1"], df["PC2"] = comp[:, 0], comp[:, 1]
var = pca.explained_variance_ratio_
print(f"Varianza explicada: PC1 = {var[0]*100:.1f}%, PC2 = {var[1]*100:.1f}%, "
      f"acumulada = {var.sum()*100:.1f}%")

plt.figure(figsize=(7, 5))
sns.scatterplot(data=df, x="PC1", y="PC2", hue="cluster",
                palette="tab10", s=45)
plt.title("Clusters proyectados sobre los 2 primeros componentes principales")
plt.tight_layout()
plt.show()

2.6 Motivación de PCA y clustering en *este* escenario

¿Por qué clustering (separación por clusters)?

El objetivo es agrupar estudiantes por rendimiento, pero no existe una variable objetivo que indique a qué grupo pertenece cada uno: es un problema no supervisado. K-Means permite descubrir perfiles naturales de estudiantes (homogéneos por dentro, distintos entre sí) para *generar hipótesis* y diseñar intervenciones diferenciadas (p. ej. apoyo al grupo "desconectado", desafíos al de "alto rendimiento"). No se busca predecir un valor, sino encontrar regularidades en los datos.

¿Por qué PCA (análisis de componentes principales)?

Las 6 variables están correlacionadas (quien estudia más suele tener mejor nota y asistencia), lo que introduce redundancia y hace imposible visualizar los grupos en 6 dimensiones. PCA condensa la información en pocos componentes no correlacionados que retienen la mayor parte de la varianza, lo que permite graficar los clusters en 2D y verificar visualmente que la separación encontrada por K-Means es real y no un artefacto. Es decir: PCA reduce y ordena el espacio; K-Means agrupa dentro de él.
